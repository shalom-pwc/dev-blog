HTML5

April 15, 2020. 3 min read . 8 min . 8 comments

HTML( Hypertext Markup Language ) is the markup language that we use to structure and give meaning to our web content, for example defining paragraphs, headings, and data tables, or embedding images and videos in the page.

An HTML element usually consists of a start tag and an end tag, with the content inserted in between: 

           
            
            <!DOCTYPE HTML>
 
            <html>
            
                <head>
            
                    <link rel="stylesheet" href="index.css">
            
                    <link href="style.css" rel="stylesheet">
            
                <link href="grid.css" rel="stylesheet">
            
                </head>
            
                <body>

                    <header>

                      <figure>

                      <img src="" alt="">

                      </figure>
                      
                      <nav>

                        <ul>
                          <li> </li>
                          <li> </li>
                          <li> </li>
                          <li> </li>
                        </ul>
                        
                      </nav>

                    </header>

                    <main>

                        <h1></h1>

                        <p></p>

                        <section>
                            <article> 
                            </article>
                        </section>
                        
                        <small>
                        <a href=""> </a>
                        </small>
                        
                    </main>

                    <footer></footer>

                </body>
            
            </html>
